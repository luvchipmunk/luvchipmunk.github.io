# 네트워크?

- ## 인터넷(Internet)

Inter - 연결하다, 묶었다 + network

=> Internet - ‘여러 개의 네트워크를 묶었다’

특징) 하나의 프로토콜 사용

-> 하나의 규칙(대화) - 인터넷(TCP/IP라는 프로토콜을 공통으로 사용한다)

- ## 인트라넷

내부의 네트워크.

역시, TCP/IP 프로토콜 사용하지만 인터넷을 통해서 접속이 불가능한 구조.

아무래도 내부 IP단에서만 접속가능하게끔하거나, 사용자 인증절차가 있을 수 있음.

- ## 엑스트라넷

인트라넷과 비슷한 의미로써 소수를 위한 내부망이지만,

클라이언트나 협력회사까지도 사용할 수도 있도록 한 것이 차이점.
