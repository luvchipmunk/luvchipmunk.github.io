# **“자료형”**

\* 참고 - 파이썬에서 사용되는 모든 자료형을 ‘객체’라고 한다.

## 숫자형

- 정수형 : 말 그대로 정수 자료형

ex ) a = 12345..

- 실수형 : 소수점이 포함된 숫자

ex ) a = 1.2 (일반적인 소수표현) / a = 4.24E10 (컴퓨터식 지수 표현 - 4.24*1010)

- 8진수 (0o123 or 0O123) 또는 16진수(0x123)로 표현하나 잘 쓰이지 않음

### **연산자**

- 제곱 : ** (ex - 5**3 = 125)
- 나눗셈

\- 나머지 반환 : % (ex - 7 % 3 = 1)

\- 몫을 반환 : // (ex - 7 // 3 = 2)

\- 결과를 반환 : / (ex - 7 / 3 = 2.3333333333333335)
